# pyeasyclean
Clean your PC using this program! (Linux only)
# Installation
## Installation on Linux
Paste this into terminal:
```
curl https://raw.githubusercontent.com/VBPROGER/pyeasyclean/main/pyeasyclean > ~/pyeasyclean && chmod +x ~/pyeasyclean
```
## Running on Linux
```
~/pyeasyclean
```
